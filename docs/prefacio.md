## Objetivos ##

## Descrição ##
Estrutura base genérica para iniciar um projeto Python.

## Escopo
- Sistema operacional Linux preferencialmente (Linux Like ou WSL);
- git
- Python 3.10+
- poetry

- bandit
- blue
- coverage
- isort
- mkdocs
- mypy
- pydocstyle
- pylint
- pytest
- safety
- taskipy
- tox
